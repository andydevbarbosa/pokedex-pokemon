## 🔎 Pokédex Web Application

Uma **Pokédex interativa desenvolvida com HTML, CSS e JavaScript**, que consome dados de uma API pública de Pokémon para exibir informações detalhadas de cada criatura.
O projeto tem como objetivo demonstrar habilidades em **desenvolvimento front-end, consumo de APIs REST e manipulação dinâmica do DOM**.

A aplicação permite que os usuários **busquem, visualizem e naveguem entre diferentes Pokémons**, apresentando dados relevantes como **nome, número na Pokédex, imagem oficial e tipo**, em uma interface simples, moderna e responsiva.

## 🚀 Tecnologias Utilizadas

* **HTML5** – Estrutura semântica da aplicação
* **CSS3** – Estilização, layout responsivo e design da interface
* **JavaScript (ES6+)** – Lógica da aplicação, manipulação do DOM e requisições assíncronas
* **PokéAPI** – Fonte de dados para informações dos Pokémons

## ⚙️ Funcionalidades

* 🔍 Busca de Pokémon por **nome ou número**
* 📋 Exibição de **informações detalhadas** do Pokémon
* 🖼️ Visualização da **imagem oficial**
* 🧭 Navegação entre Pokémons
* 📱 **Interface responsiva**, adaptada para diferentes dispositivos
* 🌐 Consumo de **API REST**

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com foco em **aprimorar habilidades em desenvolvimento web**, especialmente em:

* Consumo de APIs externas
* Manipulação dinâmica do DOM
* Estruturação de interfaces responsivas
* Organização de código em JavaScript

Além disso, serve como **projeto de portfólio**, demonstrando conhecimentos práticos em **JavaScript moderno e desenvolvimento front-end**.
